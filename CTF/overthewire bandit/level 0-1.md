La contraseña del siguiente nivel se almacena en un archivo llamado **readme** ubicado en el directorio personal. Usa esta contraseña para iniciar sesión a bandido1 usando SSH. Cada vez que encuentres una contraseña para un nivel, usa SSH (en el puerto 2220) para iniciar sesión en ese nivel y continuar el juego.

Para continuar con este nivel desde bandit0 que accedimos en el anterior nivel vamos a tener que revisar el directorio haciendo uso del comando ls, despues de eso nos encontramos con el archivo readme
