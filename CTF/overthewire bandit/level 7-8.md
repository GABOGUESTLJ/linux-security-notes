The password for the next level is stored in the file **data.txt** next to the word **millionth**

para resolver este ejercicio es tan simple como usar el comando grep para filtrar una palaabra, ya que nos die que la contraseña está al lado de la palabra millionth
el comando en este caso sería 
grep "millionth" data.txt
![[Pasted image 20260328115701.png]]
y tenemos nuestra contraseña
